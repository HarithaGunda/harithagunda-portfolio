Media view

This web application enables Exceed call centre to view policy and claim documents. Mainly pdf (created by Dialogue) and tiff (scanned post) but potentially any documents type (i.e. audio & video stream of data) can be stored in Content manager. Aviva uses Content manager as the repository of the documents; it is interfaced with Exceed system applications to create a complete Insurance processing solution. Services that are available for policy holders are Search (with Filter options such as Date, Policy Number etc.) and View. The complex interaction between application, Exceed System and Content Manager repository is through SOAP web services.
