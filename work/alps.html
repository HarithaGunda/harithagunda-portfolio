ALPS

This application replicates the workflow part of IBM Image plus ( a Legacy system used for storing Life insurance documents) functionalities with the help of a Java daemon process and stores inbound documents into Content Manager. Java Daemon will pick the inbound documents requests from shared services group transfers work basket of Content Manager (Policy or Claim correspondence documents that does not have Barcode) based on regex validations and calls the SSG Web Service which in turn notifies the indexing team.
